## 📦 The Stack

**[Agents] [Agent Safehouse – macOS-native sandboxing for local agents](https://agent-safehouse.dev/)** — A single-script, zero-dependency macOS wrapper that enforces a kernel-level deny-first sandbox around local coding agents (Claude, Codex, Gemini, etc.), blocking your `~/.ssh`, `~/.aws`, and other repos by default, with opt-in read/write grants per directory. It flips the current "agents inherit full user permissions" model on its head — you shell-alias your agent through it and forget it's there. Finally, a way to run yolo mode without the yolo.

**[Agents] [We should revisit literate programming in the agent era](https://silly.business/blog/we-should-revisit-literate-programming-in-the-agent-era/)** — The argument: the reason literate programming never took off was the cost of keeping prose and code in sync; agents eliminate that cost entirely, since they can tangle Org Mode files, update docstrings, and maintain prose-code coherence without ever getting tired. The community response confirmed real adoption — nbdev, Claude Code skills, Go + style guides, AGENTS.md files — converging on a lightweight literate-programming-adjacent practice as a legitimate agent-quality multiplier. If you're not giving your agent a narrative to work from, you're leaving output quality on the table.

**[Breakout] [Living human brain cells play DOOM on a CL1 [video]](https://github.com/SeanCole02/doom-neuron)** — Cortical Labs' CL1 platform puts 200,000 live human neurons on a chip and interfaces them with a PPO RL pipeline via electrical stimulation. The critical claim — that it's the neurons doing the learning, not the silicon decoder — is backed by ablation tests: replace CL1 spikes with random or zero signals and reward improvement flatlines. The training code is open-sourced; the architecture bridges non-differentiable spike outputs using REINFORCE-style policy gradients. Whether this scales past "neuron pong" is genuinely open, but it's the first reproducible, code-released demo of biological neural compute with verifiable neuron-side learning. 😼
