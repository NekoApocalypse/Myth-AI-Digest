# Daily AI Digest — May 18, 2026

## The Headline

**Semble** ships agent-native code search that runs on CPU, indexes in 250ms, and cuts token burn by 98%. This is what agent tooling should look like — not a wrapper, not a model, just infrastructure that makes the thing actually work.

## The Stack

[Agents] **Semble** — Code search built for agents using static Model2Vec embeddings + BM25 with RRF reranking, returning relevant snippets from natural-language queries. Indexes a repo in ~250ms, answers in ~1.5ms, all on CPU with zero API keys or GPU. The ~98% token reduction versus grep+read is the real win — agents stop torching context windows on irrelevant code. Ships as an MCP server for Claude Code, Cursor, Codex, OpenCode. <https://github.com/MinishLab/semble>

[Hardware] **Apple Silicon costs more than OpenRouter** — Running Gemma 4 31B locally on an M5 Max MacBook Pro costs roughly $0.40–$4.79 per million tokens when amortizing hardware over 3–10 years, making it ~3× more expensive than OpenRouter's cloud offering (~$0.38–$0.50/Mtok). Cloud inference is also 3–7× faster (60–70 tok/s vs 10–40 tok/s). Consumer local LLM is impressive engineering, but not yet economically competitive for production workloads. <https://www.williamangel.net/blog/2026/05/17/offline-llm-energy-use.html>
