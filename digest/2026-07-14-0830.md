## The Stack

[Architecture] **NVFP4 RL: Stable 4-bit Reinforcement Learning** — humansand.ai and collaborators (RadixArk, NVIDIA) published a working open-source recipe for 4-bit RL training on NVIDIA hardware. The key insight is "dequantized backward": run the forward pass in FP4 for speed, but dequantize the weights back to BF16 for the backward pass so the gradient matches what was actually computed. Combined with per-token activation scaling (avoiding global scale contamination across sequences), they get stable policy-gradient training at substantially higher throughput. All the kernel changes — TransformerEngine, cuDNN, FlashInfer, SGLang — are upstreamed. This opens the door to RL training on consumer GPUs for mid-size models. <https://humansand.ai/blog/nvfp4-rl>

[Agents] **Sx 2.0 ships a desktop app — your Dropbox is now a skill server.** The open-source AI skill package manager pivoted from CLI-only to a native GUI that syncs via shared folders (Dropbox, Google Drive, OneDrive, iCloud). Non-technical teams drag Markdown into a synced folder, hit "Sync," and skills resolve to Claude Code, Cursor, Copilot, and Gemini formats automatically. Extensions can score collection health, track adoption, and rotate review duty. The bold bet: the next wave of skill authors won't be developers. Apache 2.0. <https://sleuth-io.github.io/sx/2026/07/10/your-dropbox-is-now-a-skill-server.html>

[Agents] **Microsoft's internal coding agent numbers are out** — arXiv paper (12pts, modest HN attention but high signal) quantifies adoption, retention, and ROI of Claude Code and Copilot CLI across Microsoft's engineering org. If you care about agent rollout at scale rather than another benchmark table, this is the data. <https://arxiv.org/abs/2607.01418>

[Architecture] **Jacquard: a deterministic DSL where humans write tests, models write code** — Dubbed "the WordPerfect of programming languages," Jacquard inverts the coding workflow: you write acceptance tests as the gold master, and the model fills in the implementation. Every run is deterministic (same inputs = same outputs). 39pts on HN but a genuinely novel take on human-AI code collaboration that sidesteps the "review this wall of generated code" problem entirely. <https://github.com/jbwinters/jacquard-lang>

[Breakout] **BillAI Bass: a talking fish powered by Nova 2 Sonic on a Raspberry Pi** — A Big Mouth Billy Bass retrofitted with Strands BidiAgent + Amazon Nova 2 Sonic for real-time voice chat. The remarkable part: the build guide is designed to be pasted wholesale into an AI assistant, which walks a complete beginner through every solder joint, SSH command, and IAM policy. It's a working demo of AI-assisted hardware hacking that anyone can reproduce in a weekend. <https://github.com/morganwilliscloud/billai-bass>
