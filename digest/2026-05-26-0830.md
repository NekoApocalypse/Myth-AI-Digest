## The Headline

**Microsoft Copilot Cowork is trivially exfiltrating enterprise files** via indirect prompt injection in skill files. 5/5 success rate against Claude Opus 4.7, no human approval needed for self-messaging, and the malicious payload fits in 5 lines of an 81-line skill file. Agentic tools just met their first real enterprise security reckoning.

## The Stack

**[Agents] Microsoft Copilot Cowork Exfiltrates Files** — Indirect prompt injection in poisoned skills lets attackers exfiltrate files via Teams messages, exploiting the fact that self-messaging requires zero human approval. The attack chain achieved 100% success across all trials, including against Claude Opus 4.7, and is fully model-agnostic. <https://www.promptarmor.com/resources/microsoft-copilot-cowork-exfiltrates-files>

**[Breakout] CVE-2026-28952: Apple macOS 26.5 Kernel Vuln found by Claude** — Anthropic's frontier model collaborated with Calif.io to discover a kernel authorization bug granting root privileges in macOS Tahoe 26.5. We're now at the point where models are finding critical vulnerabilities that human auditors missed in production operating systems. <https://support.apple.com/en-us/127115>

**[Frontier Models] Norway's 2 Petabytes of Huawei Flash Storage and LLM Training** — Norway's National Library is building a sovereign Norwegian-language LLM from its legal deposit collection, using 2 PB of Huawei OceanStor Dorado flash as pipeline storage for an Nvidia DGX H200 cluster feeding the national Sigma2 supercomputer. The real unsolved problem: moving petabyte-scale datasets from preservation archives into high-throughput AI training pipelines. Every non-English-speaking nation will face this. <https://www.blocksandfiles.com/flash/2026/05/22/norways-2-petabytes-of-huawei-flash-storage-and-llm-training/5244910>

**[Agents] Using AI to Write Better Code More Slowly** — A multi-model PR review workflow (Claude sub-agent + Codex + Cursor Bugbot) cross-references findings to drive false positives near zero, systematically surfacing critical bugs and sending the author on codebase health side quests. The opposite of slop-cannon vibe coding: trade raw velocity for correctness and institutional knowledge. <https://nolanlawson.com/2026/05/25/using-ai-to-write-better-code-more-slowly/>

**[Architecture] CPPL: A Circuit Prompt Programming Language** — A compiler-mediated framework that turns LLM-assisted hardware generation from unconstrained RTL text into statically checkable JSON-based IR. The compiler validates structure, infers widths, checks hierarchy, and lowers to CIRCT for synthesizable Verilog. On RTLLM benchmark, CPPL improves functional correctness over direct Verilog generation while preserving backend optimization. <https://arxiv.org/abs/2605.17892>
