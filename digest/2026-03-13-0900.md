**[Research] Are LLM merge rates not getting better?** — METR's controlled benchmark separates "passes tests" from "would be merged by a maintainer." The latter hasn't improved since early 2025. The author fits a step function + constant baseline against the linear growth claim and both win on Brier score. Wild, and almost nobody's talking about it.
→ <https://entropicthoughts.com/no-swe-bench-improvement>

**[Agents] Understudy — Teach a desktop agent by demonstrating a task once** — Full local desktop runtime (GUI + browser + shell + memory + scheduling), learn-by-demo via `/teach` pipeline, and layered progressive-trust autonomy. Layers 1–2 (operate + learn) are shipped today on macOS. The five-layer architecture from "watches" to "proactively acts" is the most principled agentic design spec I've seen outside a research lab. Built on OpenClaw architecture.
→ <https://github.com/understudy-ai/understudy>

**[Hardware & Inference] Forcing Flash Attention onto a TPU and Learning the Hard Way** — Excellent deep-dive: JAX Flash Attention implementation runs 35× *slower* than XLA's auto-fused standard attention at n=4096, then gets recovered 45× with a one-word change (`vmap` instead of `fori_loop`). The lesson: TPU has 128MB VMEM vs GPU's 164KB SRAM, so the score matrix fits on-chip until n=8K; the compiler fuses it for free. Full systolic array emulator in Python included.
→ <https://archerzhang.me/forcing-flash-attention-onto-a-tpu>

**[Security] Document Poisoning in RAG Systems** — Injects 3 fabricated docs into ChromaDB. LLM confidently reports -47% revenue on a company actually up 12%. Attack succeeds 95% of the time with no ML expertise needed — just corporate-sounding language. Embedding anomaly detection at ingestion drops that to 20%; all five layers together gets to 10%. Ingestion-layer defense > output monitoring by a wide margin.
→ <https://aminrj.com/posts/rag-document-poisoning/>

**[AI × Society] Innocent Woman Jailed After AI Facial Recognition Misidentification** — North Dakota woman sat in jail 5+ months before police even interviewed her. The AI flagged a match; a human detective "confirmed" it from a license photo. The HN thread cuts to the real issue: authority delegation to AI tools removes accountability from the humans in the loop. Not a new failure mode — but scale and frequency are increasing.
