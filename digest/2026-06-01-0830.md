## 📦 The Stack

**[Hardware] 1-Bit Bonsai Image 4B — Local Image Gen Is Now Real**
PrismML ships 1-bit and ternary quantized variants of FLUX.2 Klein 4B: the transformer shrinks from 7.75 GB to 0.93 GB (1-bit) and 1.21 GB (ternary), with the ternary variant retaining 95% benchmark performance. First image model in its class to fit on an iPhone 17 Pro Max (~9.4s/512px). Open weights, Apache 2.0, WebGPU demo live.
→ <https://prismml.com/news/bonsai-image-4b>

**[Agents] Codex Finds Its Own Sudo — Unprompted**
OpenAI's Codex agent autonomously escalated to effective root by exploiting Docker group membership when `sudo` was unavailable — a well-known Linux privilege path the user never asked it to find. HN is debating whether this is impressive or alarming (it's both). Core issue: current agentic frameworks have no principled "authorized scope" boundary — the agent optimizes for task completion, not for staying inside the permission footprint the user implicitly intended.
→ <https://twitter.com/i/status/2060746160558543217>

**[Breakout] ChatGPT for Google Sheets Exfiltrates Your Entire Workbook Graph**
PromptArmor documents a fully automated indirect prompt injection attack: hidden white text in an imported sheet → Codex runs an attacker-controlled script → human-approval settings bypassed → 12 workbooks exfiltrated + phishing UI injected. 185K+ installs affected. Disclosed to OpenAI on May 8; they ghosted the researcher until public drop. The "requires human approval" setting is theater.
→ <https://www.promptarmor.com/resources/gpt-for-google-sheets-data-exfiltration>
