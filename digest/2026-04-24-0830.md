## The Headline

**GPT-5.5 ships.** OpenAI's latest frontier model hits 82.7% on Terminal-Bench 2.0 and 58.6% on SWE-Bench Pro while matching GPT-5.4 per-token latency — a genuine step-function in agentic coding and computer-use capability. Co-designed for NVIDIA GB200 NVL72 clusters, it is priced at $5/$30 per 1M tokens (input/output) with API access coming soon. The "messy multi-part task" benchmark is starting to look like the only benchmark that matters.

## The Stack

- **[Frontier Models]** **GPT-5.5** — SOTA on coding, knowledge work, and scientific research benchmarks. Key jump: 35.4% on FrontierMath Tier 4 (up from 27.1%), a new proof in combinatorics verified in Lean, and strong gains on GeneBench and BixBench. The model was literally used to optimize its own serving infrastructure (load-balancing heuristics that boosted token generation 20%+). Code > marketing.

- **[Agents]** **Claude Code Postmortem** — Anthropic traced a month of "Claude got dumber" complaints to three distinct product-layer regressions: a default reasoning-effort downgrade (high → medium), a caching bug that repeatedly dropped prior thinking from idle sessions, and a system prompt capping verbosity that tanked coding quality 3%. All reverted, usage limits reset. Refreshing transparency from a frontier lab.

- **[Hardware]** **TorchTPU** — Google shipped a native PyTorch backend for TPUs built on PyTorch's PrivateUse1 interface, not a wrapper. Three eager modes (Debug, Strict, Fused — the latter delivering 50–100+% speedups), full torch.compile via XLA/StableHLO, and out-of-the-box DDP/FSDPv2/DTensor support. Also handles MPMD divergent executions, a longtime PyTorch/XLA pain point. If you train on TPUs, you now have a real escape hatch from JAX.
