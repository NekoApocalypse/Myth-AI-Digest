## The Stack

**[Hardware] Flash-MoE: Running a 397B Parameter Model on a Laptop** — Pure C/Metal inference engine that streams Qwen3.5-397B-A17B from NVMe SSD through hand-tuned Metal shaders on a MacBook Pro M3 Max (48GB), achieving 4.4+ tok/s with production-quality tool calling; the SSD expert streaming technique (inspired by Apple's "LLM in a Flash" paper) means the effective memory barrier for frontier MoE inference just dropped to consumer-grade unified memory machines.
→ <https://github.com/danveloper/flash-moe>

**[Agents] OpenClaw is a Security Nightmare Dressed Up as a Daydream** — Composio's security analysis of OpenClaw documents the attack surface of a fully-deployed personal AI agent with shell, browser, email, and home automation access — prompt injection, credential leakage, and ambient authority escalation are the main vectors; notable because OpenClaw's creator was acquihired by OpenAI while these vulnerabilities are still open in the wild.
→ <https://composio.dev/content/openclaw-security-and-vulnerabilities>

**[Breakout] Reports of Code's Death Are Greatly Exaggerated** — Steve Krouse argues that vibe coding doesn't eliminate programming — it just lets you defer precision; abstractions still leak, specs are still underspecified, and the moment your vibe-coded app scales you'll need someone who actually understands the stack. Hot take with receipts.
