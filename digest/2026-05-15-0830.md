**Headline:** Codex goes mobile — OpenAI ships async human-in-the-loop for long-running agentic sessions, meaning you can approve, redirect, and review multi-hour coding runs from your phone while the agent keeps building. This is the UX pattern agentic work actually needs.

## The Stack

[Agents] **Codex is now in the ChatGPT mobile app** — Full mobile remote-control for Codex sessions: approve commands, review diffs, steer execution, and unblock running agents from anywhere. The relay layer keeps your machine unexposed; Hooks are now GA for custom prompt scanning and repo-level behaviors; programmatic access tokens unlock CI pipelines. <https://openai.com/index/work-with-codex-from-anywhere/>

[Hardware] **What's in a GGUF, besides the weights — and what's still missing?** — A surgical audit of the GGUF spec: what's already bundled (Jinja2 chat templates, special tokens, sampler chain ordering) and what forces every inference engine to ship its own hardcoded model-specific parsers (tool-call grammars, think tokens, multimodal projection models, feature flags). The proposed additions, if adopted, would make model-agnostic local inference libraries like NobodyWho dramatically cleaner. <https://nobodywho.ooo/posts/whats-in-a-gguf/>

[Breakout] **arXiv: 1-year ban for hallucinated references** — arXiv is enforcing a hard 1-year ban for papers with AI-fabricated citations, after which authors must pass peer review before returning. Policy is attributed to Tom Dietterich citing arXiv's Code of Conduct. The HN thread debates fraud vs. negligence, but the practical effect is clear: LLM-slop in the scientific record now has a non-trivial cost. <https://twitter.com/tdietterich/status/2055000956144935055>
