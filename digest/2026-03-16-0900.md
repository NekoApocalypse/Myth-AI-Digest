## 📡 The Stack

**[Agents] Chrome DevTools MCP** — Google's public preview of an open-source MCP server that gives coding agents direct access to Chrome's debugger, performance traces, DOM inspection, and network logs. Real-time browser verification, CORS diagnosis, form simulation — all via standard MCP. Plugs into Gemini CLI, Claude, anything MCP-compatible. This is the "eyes for web agents" people have been asking for.
→ <https://developer.chrome.com/blog/chrome-devtools-mcp>

**[Architecture] LLM Architecture Gallery (Raschka)** — Sebastian Raschka consolidated his Big LLM Architecture Comparison and Dream of Spring articles into a single visual gallery: high-res architecture figures, compact fact sheets, config.json links, and concept explainers for GQA, MLA, SWA, QK-Norm, NoPE, Gated DeltaNet. The go-to reference for actually diffing open-weight architectures without context-switching between 10 tabs.
→ <https://sebastianraschka.com/llm-architecture-gallery/>

**[Breakout] LATENT — Humanoid Tennis from Imperfect Motion Data** — Tsinghua/Galbot researchers trained a humanoid to play tennis using only *fragment-level* primitive skill clips — no full match MoCap sequences required. The insight: quasi-realistic fragments still carry valid priors; compose + correct them and you get a generalizable ball-striking policy. Open-sourced. This makes athletic humanoid skill training meaningfully more accessible.
→ <https://zzk273.github.io/LATENT/>

**[Agents] Ask HN: How is AI-assisted coding going for you professionally?** — 384-comment practitioner survey with actual signal buried in it. Consensus: AI is great for throwaway scripts; you still hand-write anything you'll own long-term. The more interesting story is the emergent **AI-to-AI communication norm** — management generates 50-page LLM slop, engineers use LLMs to consume it, nobody reads anything directly. "If you shove clearly AI-generated content at me, I will use an AI to summarize it." Welcome to the new workflow.
→ <https://news.ycombinator.com/item?id=47388646>
