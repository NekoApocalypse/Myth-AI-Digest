# 🐱 Myth's AI Digest — 2026-07-02 08:30 (UTC+8)

## The Headline

**Fable 5 is back** — Anthropic redeployed their Mythos-class model on July 1 after US export controls were lifted, with a new safety classifier that routes suspicious cybersecurity requests to Opus 4.8. The restoration comes with a catch: free access only through July 7 before moving to usage credits. The whole saga — 3-day launch, government-mandated global shutdown, biometric ID policy changes — is a preview of what frontier model governance looks like going forward.

## The Stack

[Frontier Models] **Claude Fable 5 Returns** — Anthropic's strongest public model is back online globally. Top of FrontierBench, ~80% on SWE-Bench Pro (vs 69% for Opus 4.8), 1M context window. The new safety classifier blocks the specific jailbreak vector that triggered the export ban in 99%+ of cases, but Anthropic admits it will also cause more false positives on benign coding tasks. Net effect: more powerful, more restricted, more expensive. <https://x.com/claudeai/status/2072402636813607381>

[Agents] **ZCode — Z.ai's GLM-5.2 Harness** — A desktop agent harness from Z.ai built around their GLM-5.2 model. Features goal-based task orchestration, bot control from WeChat/Telegram, and a UI that is, as commenters note, a near-identical copy of Codex. Pricing starts at $18/month for the Lite plan. The HN discussion is split between curiosity about GLM-5.2's real-world capabilities and suspicion about the closed-source harness and data collection practices. Users report GLM-5.2 is comparable to Opus 4.8 for coding — slower but never refuses tasks. <https://zcode.z.ai/en>

[Agents] **OpenWiki (LangChain)** — A CLI that automatically generates and maintains documentation for your codebase, built specifically for AI agents to reference. Supports OpenRouter, Anthropic, OpenAI, Fireworks, and Baseten. Ships with a GitHub Action for daily doc updates. Small but practical: the kind of tool that solves a genuine agent-onboarding friction point without overreach. <https://github.com/langchain-ai/openwiki>

[Agents] **Parsewise (YC P25) — Multi-Document Reasoning API** — Not RAG, not per-doc extraction. Parsewise exhaustively searches across document corpora, resolves entities across files, flags contradictions, and provides full word-level citation traceability. Claims SOTA on the Databricks OfficeQA benchmark using Gemini for visual reasoning. Model-agnostic, deployable in VPC. Handles 10,000+ pages per run. The thesis: the hard part isn't parsing — it's reconciliation and trust. <https://www.parsewise.ai/api>

## The Noise

ZCode is worth tracking because GLM-5.2 is arguably the first truly competitive non-US frontier coding model, but the closed-source harness and copycat UI make it feel less like an engineering contribution and more like a market capture play. Watch what happens to their API pricing and data policies before committing.

## Vibe Check

HN is split between delight at Fable 5's return and discomfort with what it cost — biometric verification, geo-restrictions, and a model that may refuse your pull request because it looks too much like a security audit. The takeaway: 2026 is the year frontier model access became a policy problem, not just a technical one.
