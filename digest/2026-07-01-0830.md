# Daily AI Digest — 2026-07-01

## The Headline
**Anthropic drops Claude Sonnet 5 and Claude Science in a single day**, shipping the most agentic Sonnet yet alongside a full scientific workbench — while getting caught steganographically marking Claude Code requests to catch distillers. Busy day in San Francisco.

## The Stack

**[Frontier Models] Claude Sonnet 5** — 837 pts. Anthropic's new mid-tier model narrows the gap with Opus 4.8 on agentic reasoning, tool use, and coding at roughly half the price ($2/M input, $10/M output intro pricing). Ships with cyber safeguards on by default. Scores close to Opus 4.8 on BrowseComp and OSWorld-Verified, with a much wider cost-performance Pareto frontier via adjustable effort levels. <https://www.anthropic.com/news/claude-sonnet-5>

**[Agents] Claude Code steganographically marking requests** — 1349 pts. A developer reverse-engineered Claude Code's API traffic and found it embeds environment-derived watermarks in requests — encoding shell, timezone, and platform info as steganographic markers in prompts. The likely purpose: detecting unauthorized model distillation by foreign labs. No disclosure to users. <https://thereallo.dev/blog/claude-code-prompt-steganography>

**[Agents] Claude Science** — 345 pts. A dedicated scientific workbench app that integrates 60+ curated skills and MCP connectors for genomics, single-cell, proteomics, cheminformatics. Multi-agent architecture with coordinating, specialist, and reviewer agents. Manages compute across local, HPC, and Modal GPU clusters. Generates fully reproducible, auditable artifacts with code provenance. NVIDIA BioNeMo integration included. <https://claude.com/product/claude-science>

**[Frontier Models] Nano Banana 2 Lite (Gemini 3.1 Flash-Lite Image)** — 294 pts. Google DeepMind's new cost-optimized image model trades some quality for ~2.7× faster generation than Gemini 3.1 Flash Image. Targets real-time use cases: game asset streaming, interactive design tools, agent-driven visual generation. Available via Gemini API. <https://deepmind.google/models/gemini-image/flash-lite/>

**[Architecture] TabFM: Zero-shot foundation model for tabular data** — 30 pts. Google Research's approach to treating tabular classification/regression as an in-context learning problem. Hybrid architecture with alternating row-column attention + row compression + ICL transformer. Trained entirely on hundreds of millions of synthetic datasets via structural causal models. Ships on Hugging Face and integrates into BigQuery via `AI.PREDICT` SQL. <https://research.google/blog/introducing-tabfm-a-zero-shot-foundation-model-for-tabular-data/>

**[Frontier Models] Leanstral 1.5** — 78 pts. Mistral's updated formal proof engineering model (119B total / 6.5B active parameters) optimized for Lean 4 automated theorem proving and autoformalization. 256k context. Free. <https://docs.mistral.ai/models/model-cards/leanstral-1-5-26-06>

**[Frontier Models] US lifts export controls on Claude Fable 5 and Mythos 5** — 66 pts. Department of Commerce reverses restrictions on Anthropic's top-tier models. Access restoration begins today. The models were presumably restricted over the same foreign distillation concerns that the steganography story above is about. <https://twitter.com/AnthropicAI/status/2072106151890809341>

**[Breakout] Meta Brain2Qwerty: Brain waves to words without surgery** — 98 pts. Meta AI's non-invasive MEG-based system decodes typed text from brain signals during active typing. Uses large transformers to map motor cortex signals to keystrokes. Provides code and dataset. Still requires massive helmet hardware, but eliminates surgical implants. <https://ai.meta.com/blog/brain2qwerty-brain-ai-human-communication/>

## The Noise

**Meta's brain-to-text** is running on the same "motor cortex during typing" trick that's been repackaged since BrainGate. ~75% accuracy on simple classification tasks through a multi-ton MEG machine. The press makes it sound like telepathy; the actual paper is a small incremental improvement with nice open data. Worth watching but not the breakthrough the headline suggests.

## Vibe Check

The HN thread on the steganography story hit 1349 points and dominated the discourse. The community's mood: impressed by the models, disgusted by the methods. Trust is a depreciating asset, and Anthropic's burning through it fast.
