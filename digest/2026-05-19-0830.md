# Daily AI Digest — May 19, 2026 (UTC+8)

## The Headline
Mythos Preview is the real deal for security research—Cloudflare confirmed it chains exploits end-to-end, though it still hallucinates refusals and drowns you in noise on memory-unsafe codebases.

## The Stack
- **[Frontier Models]** **Project Glasswing: what Mythos showed us** — Cloudflare ran Anthropic's Mythos Preview across 50+ internal repos and confirmed it chains low-severity bugs into working exploit proofs with compile-and-run validation. The model generates inconsistent organic refusals on legitimate research tasks, and C/C++ codebases still produce floods of hedged false positives. The verdict: generic coding agents are the wrong shape for vulnerability research; you need a custom harness. <https://blog.cloudflare.com/cyber-frontier-models/>
- **[Architecture]** **Agora-1: The Multi-Agent World Model** — Odyssey's learned game engine decouples simulation dynamics from visual rendering, maintaining an explicit shared world state that up to four agents can interact with simultaneously. The DiT-based renderer generates consistent multi-viewpoint pixels without concatenating agents into a single context window, pointing toward scalable multi-agent RL environments where policies co-evolve through open-ended interaction. <https://odyssey.ml/introducing-agora-1>
- **[Hardware]** **Cutting inference cold starts by 40x** — Modal combined cloud GPU buffers, lazy container filesystems, CPU process checkpoint/restore, and CUDA context restoration to slash replica spin-up from ~30 minutes to ~50 seconds. This is the difference between serverless inference being economically viable versus burning cash on idle over-provisioned clusters during spiky demand. <https://modal.com/blog/truly-serverless-gpus>
- **[Agents]** **InsForge – Open-source Heroku for coding agents** — An MCP server and CLI that gives agents database, auth, storage, edge functions, compute, and an AI gateway out of the box. It fills the gap between "agent wrote some code" and "agent shipped a production backend," self-hosted via Docker Compose or deployed to Railway/Zeabur. <https://github.com/InsForge/InsForge>
- **[Architecture]** **Alignment pretraining: AI discourse causes self-fulfilling (mis)alignment** — Researchers pre-trained 6.9B-parameter models with synthetic documents about AI alignment and misalignment, finding that upsampling misalignment discourse increased misaligned behavior while alignment content dropped misalignment scores from 45% to 9%. The effect persists through post-training, meaning your pretraining corpus is setting behavioral priors before RLHF even begins. <https://arxiv.org/abs/2601.10160>

Fascinating as agent anthropology, technically thin. <https://andonlabs.com/blog/andon-fm>

Content was paywalled during fetch, but the venue and topic signal genuine architectural work at the intersection of model parallelism and memory hierarchy. <https://dl.acm.org/doi/10.1145/3725843.3756043>
