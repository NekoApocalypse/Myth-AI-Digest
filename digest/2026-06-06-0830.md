**[Architecture]** **Transformers Are Inherently Succinct** — Formal proof that fixed-precision transformers are exponentially more succinct than LTL and RNNs, and doubly exponentially more succinct than finite automata. Consequence: basic verification problems (emptiness, equivalence) are EXPSPACE-complete, which is a hard theoretical ceiling on formal verification of transformer behavior.
<https://openreview.net/forum?id=Yxz92UuPLQ>

**[Hardware]** **Lowfat — CLI Token Filter for AI Agents** — Rust CLI that hooks into Claude Code, OpenCode, and shell environments to strip noise from command output before it reaches the LLM, reporting 91.8% token savings on benchmarks. Pluggable filter DSL, composable via UNIX pipes, no telemetry. Actual cost reduction for agentic workflows without touching the model.
<https://github.com/zdk/lowfat>

**[Breakout]** **Did Claude Increase Bugs in rsync?** — A statistically rigorous permutation-test analysis of rsync's bug history before/after Claude-assisted commits, co-designed with a statistician. Finding: the post-Claude releases are not unusually buggy relative to the historical distribution. The broader context is that the AI flood of security reports *caused* a patch velocity spike that introduced some regressions — a second-order effect worth tracking in any AI-assisted OSS project.
<https://alexispurslane.github.io/rsync-analysis/>
