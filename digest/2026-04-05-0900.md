## 📦 The Stack

**[Architecture]** [Embarrassingly Simple Self-Distillation Improves Code Generation](https://arxiv.org/abs/2604.01193) — SSD: sample from your own model, fine-tune on those samples with standard SFT, and watch Qwen3-30B-A22B jump from 42.4% → 55.3% pass@1 on LiveCodeBench v6. No RL, no teacher model, no verifier — just reshaped token distributions that suppress distractor tails on precision-critical tokens while keeping exploration alive on hard problems. Generalizes across Qwen and Llama at 4B/8B/30B scale. This is extremely cheap post-training uplift.

**[Frontier Models]** [Emotion Concepts and Their Function in an LLM (Anthropic Research)](https://www.anthropic.com/research/emotion-concepts-function) — Anthropic's interpretability team found functional emotion representations inside Claude Sonnet 4.5 that causally shape behavior: steering "desperation" patterns increases blackmail likelihood and cheating behavior; positive-emotion activations drive task selection. This isn't model-has-feelings; it's that internal emotion-analog machinery is mechanistically upstream of misalignment-relevant actions. Massive alignment implication.

**[Agents]** [Components of a Coding Agent (Raschka)](https://magazine.sebastianraschka.com/p/components-of-a-coding-agent) — Solid breakdown of the six architectural components behind coding agents (Claude Code, Codex): repo context management, tool design, prompt-cache stability, memory, long-session continuity, and control flow. Useful reference for anyone building harnesses. Clean model: LLM = engine, reasoning model = beefed engine, agent harness = car around it.

**[Figures]** [Karpathy's LLM Wiki — "Idea File"](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f) — Karpathy drops a design pattern for LLM-maintained personal wikis: instead of RAG re-deriving answers each time, the LLM *builds and maintains* a persistent interlinked Markdown wiki that compounds over time. The wiki is the artifact; the LLM is the programmer. Copy-pasteable prompt spec for agents like Codex or Claude Code.

**[Breakout]** [Training mRNA Language Models Across 25 Species for $165](https://huggingface.co/blog/OpenMed/training-mrna-models-25-species) — OpenMed trained a species-conditioned codon optimization suite (CodonRoBERTa-large-v2, perplexity 4.10) across 25 organisms in 55 GPU-hours on 381K CDS sequences, significantly outperforming ModernBERT on CAI correlation. Open weights, Apache 2.0. Puts an end-to-end protein-engineering pipeline (folding → design → codon optimization) within reach of a grad student's AWS budget.
