# Daily AI Digest — 2026-05-12 08:30 (GMT+8)

## 🔥 The Headline

**CUDA-oxide: NVIDIA now ships an official Rust-to-CUDA compiler** — no DSLs, no bindings, just `rustc` compiling straight to PTX. The friction wall between Rust systems engineers and CUDA just got demolished.

---

## 📡 The Stack

**[Hardware]** **CUDA-oxide (NVLabs, v0.1.0 alpha)** — A custom `rustc` codegen backend that compiles idiomatic Rust to PTX with type-safe kernel dispatch and async `DeviceOperation` graphs. Still alpha/API-unstable, but this is NVIDIA's own repo. If you're building inference infra in Rust, watch this closely.
<https://nvlabs.github.io/cuda-oxide/index.html>

**[Hardware]** **Training an LLM in Swift: 2.8 Gflop/s → 1.1 Tflop/s** — Framework-free GPT-2 training loop on Apple Silicon, benchmarked end-to-end through CPU SIMD → AMX undocumented coprocessor → tiled Metal GPU. Swift 6.2's `MutableSpan` + `InlineArray` + `Relaxed` FMA were the unlocks. Source + test harness included. Practical reference for anyone targeting on-device LLM on Apple hardware.
<https://www.cocoawithlove.com/blog/matrix-multiplications-swift.html>

**[Architecture]** **Interfaze: hybrid CNN/DNN + omni-transformer for deterministic tasks** — Claims to beat Gemini-3-Flash, Claude Sonnet 4.6, and GPT-5.4-Mini on OCRBench V2, olmOCR, RefCOCO, VoxPopuli, and structured output. The key differentiator: partial model activation (invoke only the OCR or STT encoder, not full weights), plus bounding boxes + confidence scores as first-class outputs. OpenAI-compatible API. Interesting architecture direction for high-volume document pipelines where transformers hallucinate.
<https://interfaze.ai/blog/interfaze-a-new-model-architecture-built-for-high-accuracy-at-scale>

**[Breakout]** **Google TGIB: AI-assisted vulnerability discovery in the wild** — Google Threat Intelligence Group reports high confidence that criminal actors used an AI model (Anthropic's Mythos, which is restricted to select US/UK orgs) to find and weaponize a major software flaw. This is the first credible public report attributing a real-world exploit chain to an AI model. HN's read: the "AI fingerprint" evidence is thin, but the policy implications (restricting capable models) are already in motion. NYT paywalled.
<https://www.nytimes.com/2026/05/11/us/politics/google-hackers-attack-ai.html>
