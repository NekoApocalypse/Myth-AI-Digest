Total component spending grew from ~$22B (2024) to ~$52B (2025), with HBM alone accounting for ~$20B of that increase. Logic die share stayed flat at ~13-14%. The memory wall is the cost wall. <https://epoch.ai/data-insights/ai-chip-component-cost-shares>

[Architecture] **Constraint Decay: The Fragility of LLM Agents in Backend Code Generation** — Systematic study across 80 greenfield + 20 feature tasks in eight web frameworks finds "constraint decay": as structural requirements accumulate, agent assertion pass rates drop ~30 points. Agents succeed in minimal frameworks (Flask) but crater in convention-heavy ones (FastAPI, Django). Root cause: data-layer defects (bad queries, ORM violations). Jointly satisfying functional + structural requirements remains unsolved. <https://arxiv.org/abs/2605.06445>

[Frontier Models] **Apple's PICO (Perceptual Image Codec)** — First learned codec optimized directly for human visual perception via large-scale subjective studies. 2.3-3× bitrate savings vs AV1/AV2/VVC/ECM/JPEG-AI, 20-40% vs other learned codecs. On iPhone 17 Pro Max: 230ms encode, 150ms decode for 12MP images — faster than most ML codecs on a V100 GPU. Cross-platform robustness guarantees included. 🍎 doing actual ML engineering again. <https://apple.github.io/ml-pico/>
