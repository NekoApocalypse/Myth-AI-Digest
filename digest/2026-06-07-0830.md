## 📦 The Stack

**[Agents] Harness Engineering: Leveraging Codex in an Agent-First World**
OpenAI's internal team built a production product with zero manually-written code over 5 months — GPT-5-powered Codex handled application logic, tests, CI, docs, and observability. Human engineers only wrote prompts, broke down goals, and wired Chrome DevTools Protocol + ephemeral observability stacks directly into the agent runtime so Codex could reproduce bugs and validate fixes without human copy-paste. Agent-to-agent PR review replaced most human review. 3.5 PRs/engineer/day; throughput increased as team grew from 3 to 7.
<https://openai.com/index/harness-engineering/>

**[Frontier Models] Benchmarks in Leipzig**
49 mathematicians spent 3 days at Max Planck building 100 research-level math questions. Stage 1: 5 SOTA LLMs attempted each once — 41 questions remained completely unsolved. Stage 2: 20 runs per model — dropped to 16 unsolved. Stage 3: heavy-thinking models — only 2 unsolved. Frontier math reasoning is now operating at research-mathematician level on ~98% of expert-curated problems.
<https://arxiv.org/abs/2606.05818>

**[Breakout] Meta Confirms 20,225 Instagram Accounts Hacked via AI Chatbot**
Meta's AI-assisted account recovery chatbot had a missing email-ownership check in a separate code path — attackers could redirect password-reset codes to any email just by asking. The chatbot worked as intended; the failure was an unvalidated assumption upstream. 7-week exposure window, April–May 2026. Meta disabled the chatbot and is auditing other AI support flows across platforms.
<https://this.weekinsecurity.com/meta-confirms-thousands-of-instagram-accounts-were-hacked-by-abusing-its-ai-chatbot/>

**[Agents] Running Python in a MicroPython/WASM Sandbox**
Simon Willison released `micropython-wasm` (alpha): bundles a WASM MicroPython build with `wasmtime` for fully sandboxed Python execution inside Python applications. Pure PyPI install, memory/CPU limits enforced, no filesystem or network access without explicit host grants. Built specifically to power `datasette-agent-micropython` — safe LLM-driven code execution in Datasette Agent. This is the cleanest "agent executes code safely" primitive I've seen.
<https://simonwillison.net/2026/Jun/6/micropython-in-a-sandbox/>
