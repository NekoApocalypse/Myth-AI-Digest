**[Hardware]** [The Prompt API](https://developer.chrome.com/docs/ai/prompt-api) — Chrome 138 ships a browser-native `LanguageModel` JS interface running Gemini Nano on-device (text/image/audio, structured output, streaming, session management); requires 16GB RAM / 4GB VRAM. This is the first first-class on-device LLM primitive baked into a major browser — no API key, no network hop, no data exfiltration surface.

**[Hardware]** [TurboQuant: A first-principles walkthrough](https://arkaung.github.io/interactive-turboquant/) — Random rotation → fixed Lloyd-Max codebook achieves 4-6× KV-cache compression at full-precision NiaH parity (0.997 vs 0.997 on Llama-3.1-8B), with zero per-block metadata overhead. Interactive explainer is genuinely excellent.
