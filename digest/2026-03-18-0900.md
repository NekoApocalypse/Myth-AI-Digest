## The Stack

**[Frontier Models] Mistral AI Releases Forge** — An end-to-end platform for enterprises to train frontier-grade models on internal data (codebases, compliance docs, operational records), with RL alignment and evaluation loops. Supports dense + MoE architectures, and is explicitly designed to be orchestrated by autonomous agents (Mistral Vibe can run the whole fine-tune pipeline in plain English). The differentiation from generic RAG: the model *internalizes* institutional knowledge at weight level, not retrieval time.

**[Agents] Get Shit Done (GSD) — Meta-Prompting + Spec-Driven Dev System** — A context engineering layer for Claude Code (and OpenCode, Gemini CLI, Codex, Copilot) that addresses context rot by orchestrating parallel subagents in fresh 200k-token contexts per task. The key mechanism: XML-structured plans with built-in verification steps, wave-based parallel execution, and atomic commits per task. 32k+ stars. It's not novel research — it's a system that makes agentic coding actually reliable at scale.

**[Hardware] Unsloth Studio** — Desktop-native (Mac + Windows, 100% offline) app for local LLM inference and no-code fine-tuning. Runs GGUF/Safetensors models with tool-calling, web search, and OpenAI-compatible API. Data Recipes feature converts unstructured PDFs/CSVs into training datasets via a graph-node workflow. The GGUF export path (→ llama.cpp, vLLM, Ollama) makes this a serious local-first fine-tuning pipeline for practitioners.
