## 📦 The Stack

**[Agents] Claude Code: Channels** — Claude Code gets an MCP-based push channel system (Telegram/Discord in research preview); Claude can now react to CI results, chat messages, and external events while you're away. This is agentic-inbox, not chat — and it ships two-way. Real sessions, real side effects.
→ https://code.claude.com/docs/en/channels

**[Agents] Scaling Karpathy's Autoresearch to a GPU Cluster** — SkyPilot ran Claude Code against Karpathy's `autoresearch` on 16 GPUs in parallel, submitting ~910 experiments in 8 hours, cutting val_bpb from 1.003 → 0.974. The key finding: parallel search catches parameter interaction effects that sequential greedy search structurally cannot. The agent also self-discovered hardware heterogeneity (H100 vs H200) and exploited it.
→ https://blog.skypilot.co/scaling-autoresearch/

**[Architecture] NanoGPT Slowrun: 10x Data Efficiency** — An ensemble of 1.8B-parameter models (18B total) trained on 100M tokens matches standard baselines needing 1B tokens. Mechanism: chain distillation + ensembling reverses overfitting dynamics — individual models get worse past their peak, but the ensemble improves. This is a direct attack on data-wall scaling constraints.
→ https://qlabs.sh/10x

**[Hardware] KittenTTS — SOTA TTS under 25MB** — Three new open-source ONNX TTS models (15M/40M/80M params, 25–80MB), CPU-only, 24kHz output, 8 voices. The int8 nano fits in a mobile app. Removes the last excuse for cloud TTS dependency on constrained devices.
→ https://github.com/KittenML/KittenTTS

**[Architecture] EsoLang-Bench — Frontier models collapse on esoteric languages** — 80 problems across Brainfuck, Befunge, Whitespace, Unlambda, Shakespeare. Best model: 3.8% overall vs ~90% on Python. All models score 0% above Easy tier. Whitespace: permanently 0%. The benchmark validates what many suspected — high Python scores are largely memorization proxies, not reasoning.
→ https://esolang-bench.vercel.app/
