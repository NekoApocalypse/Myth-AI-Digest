## 📦 The Stack

**[Hardware] [Taalas HC1 — 17k tokens/sec on Llama 3.1 8B](https://taalas.com/the-path-to-ubiquitous-ai/)** — Custom silicon that hard-wires the model weights directly into the chip, merging storage and compute at DRAM density. Eliminates HBM, liquid cooling, and advanced packaging entirely. 10x faster than SOTA, 20x cheaper to build, 10x lower power — delivered by 24 people on $30M. A second-gen chip targeting frontier models arrives this winter. 😼

**[Hardware] [ggml.ai → Hugging Face](https://github.com/ggml-org/llama.cpp/discussions/19759)** — Full-time maintainership of llama.cpp and ggml is now HF-backed. Projects stay 100% open-source and community-driven. Near-term focus: seamless `transformers` integration and better GGUF compatibility. If you depend on local inference for anything, this is good news.

**[Agents] [Claude Code Security — agentic vuln scanning](https://www.anthropic.com/news/claude-code-security)** — Built on Claude Opus 4.6, it already found 500+ novel vulnerabilities in production open-source codebases — bugs that survived years of expert review. It reasons about code like a human researcher (data flow, component interaction), not pattern-matching like a linter. Limited preview, human approval required for all patches. This is the first credible "AI as security engineer" deployment from a frontier lab.
