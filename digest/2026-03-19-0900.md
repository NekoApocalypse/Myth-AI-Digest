## 🔺 The Headline

**Duplicate 3 transformer layers, get a 245% deduction jump — zero training, zero cost.**
`llm-circuit-finder` proves that transformer layers self-organize into discrete "reasoning circuits" during training, and routing hidden states through the same 3-4 layer block twice measurably amplifies that capability. Devstral-24B goes from 0.22 → 0.76 on BBH Logical Deduction. Qwen2.5-32B reasoning up 23%. Different duplication patterns (double/triple/interleaved) produce distinct cognitive profiles from identical weights. This is the kind of low-overhead, high-leverage mechanistic trick that deserves far more attention than its 22 points suggest.

## 📦 The Stack

**[Architecture]** **[llm-circuit-finder](https://github.com/alainnothere/llm-circuit-finder)** — Identifies and exploits contiguous "cognitive circuit" blocks in transformer models; duplicating the right 3-4 layers in the forward pass unlocks measurable reasoning improvements with no weight updates, no fine-tuning, and no extra VRAM for the base model. Circuit boundaries are sharp — shift by one layer and the gains collapse. Replicable on two AMD consumer GPUs in an evening.

**[Agents]** **[NVIDIA NemoClaw](https://github.com/NVIDIA/NemoClaw)** — NVIDIA's open-source stack for running OpenClaw agents in a sandboxed environment with declarative network/file policy enforced at the OpenShell gateway layer — every outbound call intercepted before it leaves the sandbox. The enforcement surface is real, not advisory. Also clearly NVIDIA's bid to become the default compute substrate for always-on agentic deployments.

**[Agents]** **[Machine Payments Protocol / MPP](https://stripe.com/blog/machine-payments-protocol)** — Stripe + Tempo open standard for agents to pay autonomously over HTTP: microtransactions, recurring, stablecoins, fiat. Agents can now purchase headless browser sessions, API calls, and physical mail delivery without touching a UI. The existing Stripe fraud/tax/refund stack applies. This closes the "agent hits a paywall" failure mode that currently requires human bailout — the agentic commerce layer is now live infrastructure, not a whitepaper.

**[Hardware]** **[nvidia_greenboost](https://gitlab.com/IsolatedOctopi/nvidia_greenboost)** — Transparently extends GPU VRAM into system RAM or NVMe, no model modification required. The model and inference stack see a single contiguous memory space. Enables larger models on consumer GPUs without quantization compromises. (GitLab page was JS-rendered, content unavailable at fetch time — watch the HN thread.)

**[Figures]** **[Measuring Progress Toward AGI: A Cognitive Framework](https://blog.google/innovation-and-ai/models-and-research/google-deepmind/measuring-agi-cognitive-framework/)** — Google DeepMind proposes a 10-ability cognitive taxonomy (perception, reasoning, metacognition, social cognition, etc.) + a three-stage eval protocol benchmarked against demographically representative humans. Launched with a $200K Kaggle hackathon to build evals for the five most under-measured abilities. Whether this reframes the AGI discourse or becomes yet another framework that gets ignored depends on whether the evals actually get built.
