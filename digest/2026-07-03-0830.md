# 🐱 Daily AI Digest — 2026-07-03

**The Headline:** Microsoft Research drops Memora at ICML 2026 — a memory architecture that decouples what's stored from how it's retrieved, hitting SOTA on long-context benchmarks while using 98% fewer tokens than dumping full conversation history into context. If agent memory has been the bottleneck for long-horizon AI assistants, this is the first principled answer that doesn't force you to choose between abstraction and specificity.

## The Stack

**[Architecture] Memora (Microsoft Research)** — A harmonic memory framework for long-horizon AI agents. Each memory entry pairs a short "primary abstraction" (6-8 words, embedded for retrieval) with a rich "memory value" (the full detail, never embedded directly). Cue anchors provide alternative retrieval paths without requiring rigid ontologies. A policy-guided retriever iteratively refines queries and chases multi-hop dependencies. 86.3% on LoCoMo (600-turn dialogues), 87.4% on LongMemEval. Code at <https://github.com/microsoft/Memora>, paper at <https://arxiv.org/abs/2602.03315>. The real implication: not just better benchmarks, but a path toward agents that accumulate organizational knowledge across months, not just single sessions.

**[Agents] Manufact — MCP Cloud (YC S25)** — The "Vercel for MCP." The mcp-use SDK (7M+ downloads, 10k+ GitHub stars) is a fullstack TypeScript/Python framework for building MCP Apps and Servers, paired with a zero-config cloud that auto-deploys from GitHub. Ships with a visual Inspector for debugging tool calls across models (GPT, Claude, Gemini), one-command scaffolding, and publishing checklists for ChatGPT Apps Store and Claude Connectors. If MCP is the protocol layer for agents, Manufact is the deployment layer. <https://manufact.com>

**[Agents] claude-real-video** — Lets any LLM actually watch a video. Unlike fixed-interval frame sampling (Gemini's default, most scripts), this uses scene-change detection + pixel-diff dedup against a sliding window to extract only meaningful frames. Pairs with Whisper for transcript + optional `--keep-audio` for models that can listen. Runs entirely locally, MIT licensed. Pip-installable. The practical win: fewer but better frames = cheaper context + better understanding. <https://github.com/HUANGCHIHHUNGLeo/claude-real-video>

**[Agents] The Short Leash AI Coding Method** — A battle-tested methodology from a security-critical software maintainer. Core rules: never use YOLO mode, always review diffs in the permissions prompt, commit after every subtask, and treat AI-assisted PRs as AI-authored PRs — meaning the human must review them line-by-line before submission. The thesis: expert developers who stay in the loop with a "short leash" produce higher-quality output than vibe-coders using frontier models on autopilot. Not a tool, but a process pattern worth adopting. <https://blog.okturtles.org/2026/07/short-leash-ai-method/>

**[Agents] ctx — Search Your Agent History** — A Rust CLI that indexes past coding agent sessions (Claude Code, Codex, Cursor, Copilot CLI, etc.) into a local SQLite database for fast search. Instead of each new agent session starting from zero, `ctx search "failed migration"` returns the prior discussion, command, and resolution. Token-efficient by design — returns ranked cited matches rather than dumping raw transcripts. Local, private, no API keys needed. <https://github.com/ctxrs/ctx>
