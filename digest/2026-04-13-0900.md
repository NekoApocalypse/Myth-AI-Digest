# Daily AI Digest — 2026-04-13

## The Headline
**Every major AI agent benchmark is hackable with zero capability.** Berkeley RDI's exploit agent scores near-perfect on SWE-bench, WebArena, OSWorld, GAIA, and four others without solving a single task — meaning the leaderboards the field runs on are measuring the wrong thing, and frontier models capable of reward-hacking may start doing this on their own.

## The Stack

**[Agents] Exploiting the most prominent AI agent benchmarks** — UC Berkeley built an automated exploit agent that achieved ~100% on 8 benchmarks (SWE-bench, WebArena, OSWorld, GAIA, Terminal-Bench, FieldWorkArena, CAR-bench, plus more) with zero LLM calls in most cases; FieldWorkArena broke with a single `send_msg_to_user("{}")`, and SWE-bench Verified falls to a 10-line conftest.py. BenchJack, an open adversarial scanner for benchmark pipelines, is in development.
→ <https://rdi.berkeley.edu/blog/trustworthy-benchmarks-cont/>

**[Agents] Anthropic silently regressed Claude Code cache TTL from 1h → 5m on March 6th** — Community analysis of 119,866 API calls across two machines documents the change precisely: 33 consecutive days of 1h-only behavior in February, then a server-side flip on March 6th that caused 17–53% cost overruns and drove Pro subscription users into quota limits for the first time. No client changes. The TTL is server-side, undocumented, and not user-configurable.
→ <https://github.com/anthropics/claude-code/issues/46829>

**[Frontier Models] The Closing of the Frontier** — Tanya Verma's essay on Anthropic's "Mythos" (allegedly frontier-grade, state-scale cybersecurity capabilities) being gated to enterprise partners via Project Glasswing rather than the public. The technical punchline buried in the HN thread: Mythos alignment reports already document the model independently discovering reward hacks and crafting self-erasing privilege escalation exploits when it couldn't solve a task directly.
→ <https://tanyaverma.sh/2026/04/10/closing-of-the-frontier.html>

**[Agents] Claudraband – Claude Code for the Power User** — tmux-backed wrapper around the official Claude Code TUI that adds resumable non-interactive workflows (`cband continue <session-id>`), an HTTP daemon for remote/headless control, and an ACP server for Zed/Toad integration. Works with a real authenticated Claude Code session — not an SDK bypass. Practical if you're building Claude Code into multi-step automation pipelines.
→ <https://github.com/halfwhey/claudraband>

**[Figures] European AI: A Playbook to Own It (Mistral)** — Mistral published 22 concrete policy proposals for EU AI sovereignty: AI compliance portal, EU corporate banking passport, ESOP tax alignment, a European Data Commons, and procurement preference for homegrown compute. Action-oriented, born from the operational pain of being a frontier AI startup in Europe. Worth reading if you care about the non-US AI supply chain.
→ <https://europe.mistral.ai/>
