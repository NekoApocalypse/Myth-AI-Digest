## 📦 The Stack

**[Agents] Claude Code Routines** — Schedule, API-trigger, or GitHub webhook routines that run on Anthropic's cloud with your repos and MCP connectors pre-wired. No laptop, no DIY cron. Limits: 5/15/25 runs/day on Pro/Max/Team; extra usage available. This is Anthropic owning the automation layer of the dev workflow.
→ <https://claude.com/blog/introducing-routines-in-claude-code>

**[Architecture] I-DLM: Introspective Diffusion Language Models** — First diffusion LM to match same-scale AR quality: 8B beats LLaDA-2.1-mini 16B by +26 pts on AIME-24, +15 on LiveCodeBench-v6, at 2.9–4.1x throughput. The trick: Introspective Strided Decoding (ISD) generates and verifies tokens in the same forward pass using a p/q acceptance criterion. Drop-in with SGLang; gated LoRA variant is bit-for-bit lossless. The "DLMs can't match AR quality" assumption may be dead.
→ <https://introspective-diffusion.github.io>

**[Agents] The M×N Problem of Tool Calling and Open-Source Models** — Rémi Louf (Outlines/dottxt) writes up the systemic mess: M inference engines × N model families, each independently re-implementing wire format parsers from scratch. gpt-oss Harmony, DeepSeek, and GLM5 all use incompatible token/argument serialization schemes. His proposed fix: declarative per-model format specs, like HuggingFace chat templates but for tool-call wire formats. If you run local agents on open models, you've hit this wall.
→ <https://www.thetypicalset.com/blog/grammar-parser-maintenance-contract>
