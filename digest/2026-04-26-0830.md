- **[Architecture]** **Hash Layers & Staircase Attention** — Meta AI researchers (ParlAI) demonstrate that parameter count and computation are orthogonal axes in Transformer design, not a single tied variable. Hash Layers use token hashing to route sparse mixture-of-experts without proportional compute increases, while Staircase Attention recycles the same parameters recurrently to increase compute per parameter without growing model size. The improvements are orthogonal and stackable; code is available. This is the kind of structural thinking that actually moves efficiency forward instead of just throwing more FLOPs at the problem.
