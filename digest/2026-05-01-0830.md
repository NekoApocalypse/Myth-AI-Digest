## 📡 The Stack

[Frontier Models] **Claude Opus 4.7 de-anonymizes writers from unpublished text** — Kelsey Piper fed 125 words of an unpublished column into Opus 4.7 in incognito mode; the model named her correctly. At 1,000 words it identified her from an unpublished *novel*. This is a latent capability that's been quietly lurking — Anthropic's thinking models are now strong enough stylometric profilers to threaten pseudonymous authorship at scale.
→ <https://theargumentmag.com/> | via <https://news.ycombinator.com/item?id=44092578>

[Agents] **Claude Code / OpenClaw billing enforcement is live — and model-level** — Theo (t3.gg) confirmed that a repo with `openclaw` in a JSON blob causes Claude Code to either refuse requests or route to paid "Extra Usage." Boris Cherny (Anthropic) announced the policy shift: Claude subscriptions no longer cover third-party harnesses. The enforcement mechanism is the model reading commit history — not an API gate.
→ <https://x.com/theo/status/2049645973350363168>

[Agents] **Mozilla formally opposes Chrome's Prompt API** — Mozilla's Jake Archibald posted that the Prompt API has "severe negative consequences to interoperability, updatability, and neutrality of the web platform." Core objection: developers will tune to Gemini Nano's quirks and policies, recreating the browser-compatibility problem but for AI, and Google's T&Cs (not just law) would govern web API behavior. Already shipping in Chrome + Edge.
→ <https://github.com/mozilla/standards-positions/issues/1213>

[Agents] **pu.sh — full coding-agent harness in 400 lines of shell** — No npm, no pip, no Docker. Just curl, awk, and an API key. Self-described "slop cannon" that gives you a local coding agent from a shell script. Low points (62) but the approach is worth noting: minimum-dependency agent infrastructure.
→ <https://pu.dev>
