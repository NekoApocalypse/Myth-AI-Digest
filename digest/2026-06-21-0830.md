[Agents] **Temporary Cloudflare accounts for AI agents** — Cloudflare shipped `wrangler deploy --temporary`, giving agents a 60-minute throwaway account to deploy Workers with zero human OAuth/MFA. Agent runs `wrangler`, sees the `--temporary` flag prompt, redeploys, and iterates. Human claims the account later via link. This is infrastructure catching up to the agent workflow — auth gates were the single biggest friction point for headless deployments. <https://blog.cloudflare.com/temporary-accounts/>

[Agents] **ArgusRed — post-trained pen testing model** — Cosine trained a model specifically for offensive security, not an off-the-shelf API with a prompt wrapper. Scan mode is read-only (Go harness blocks mutating tool calls deterministically). Pen test mode runs exploits in Docker sandboxes against authorized targets. Safety via infrastructure guardrails, not model refusals — the right pattern for agents that need to do dangerous things safely. <https://www.argusred.com/cli>

[Hardware] **Inference cost at scale with napkin math** — Clean first-principles walkthrough of GPU inference economics: B200 specs, matmul cost, KV-cache mechanics, GQA, duty cycle. End result: a 32B dense model on one B200 serves ~300-800 users at ~$0.013/hr per user ($9.36/mo). Framework is transferable to any model/hardware. Worth bookmarking if you're pricing an inference product. <https://injuly.in/blog/napkin-inference-cost/index.html>
