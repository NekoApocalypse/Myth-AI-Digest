# Daily AI Digest — April 29, 2026

## The Headline

Talkie-1930-13b demonstrates that training language models exclusively on historical text opens a new research frontier for contamination-free generalization experiments — and reveals how much modern web data homogenizes LM behavior.

## The Stack

- **[Breakout] Talkie: a 13B vintage language model from 1930** — A 13B LM trained on 260B tokens of pre-1931 English text enables unique experiments in temporal generalization, from forecasting historical events to learning Python without ever seeing a computer. The team plans to scale to GPT-3.5-level capability this summer. <https://talkie-lm.com/introducing-talkie>

- **[Frontier Models] VibeVoice: Open-source frontier voice AI** — Microsoft's voice model family includes a 7B ASR model that processes 60-minute audio in a single pass with speaker diarization and timestamping, and a 0.5B real-time TTS model with ~300ms latency. Both use a next-token diffusion architecture with continuous speech tokenizers at 7.5 Hz. Weights and fine-tuning code are on Hugging Face. <https://github.com/microsoft/VibeVoice>

- **[Agents] A playable DOOM MCP app** — A developer shipped DOOM as an inline MCP app that runs inside ChatGPT and Claude, proving the protocol can handle genuine interactive UI surfaces beyond structured JSON tool calls. Uses a signed-token session model that works across both inline rendering and fallback browser modes. <https://chrisnager.com/blog/doom-runs-in-chatgpt-and-claude/>
