# Daily AI Digest — 2026-04-03

**The Headline:** AMD drops Lemonade, an open-source local LLM server with native GPU+NPU co-execution — the inference access story is no longer just NVIDIA's to tell.

---

## The Stack

**[Hardware] Lemonade by AMD — open-source local LLM server (GPU + NPU)** — 436 pts
AMD enters the local inference runtime space directly: Lemonade is a 2MB native C++ service that auto-configures for your hardware, runs llama.cpp / Ryzen AI SW / FastFlowLM, and exposes a unified OpenAI-compatible API for chat, vision, image gen, and speech. NPU co-execution is the differentiator — this is AMD making a serious play to be a credible local inference platform, not just a cheaper GPU.

**[Frontier Models] Google releases Gemma 4 open models** — 1107 pts
Gemma 4 launches a family from E2B (mobile/IoT) to 31B with native thinking, hitting AIME 2026 at 89.2% and GPQA Diamond at 84.3% on the flagship. The 26B MoE variant (A4B active) scores 73.8% on MMMU Pro. Multimodal, 140-language, function-calling built in. DeepMind is clearly distilling Gemini 3 research into accessible open weights — this competes directly with Qwen 2.5 and Mistral at the efficiency tier.

**[Agents] Cursor 3 — unified multi-agent workspace** — 272 pts
Rebuilt from scratch (not a VS Code fork), Cursor 3 introduces a multi-repo, multi-agent interface with seamless local↔cloud handoff, parallel agent execution, and a plugin marketplace (MCP, subagents, skills). Cloud agents generate demos/screenshots for verification. The abstraction level shift — engineers manage fleets, not files — is the story here.

**[Frontier Models / Agents] Qwen3.6-Plus: Towards real world agents** — 421 pts
Qwen3.6-Plus is Alibaba's hosted-only frontier model targeting agentic real-world tasks (τ2-bench Retail: 86.4%). Not open weights — smaller variants promised "eventually." Community sentiment is mixed: strong benchmark numbers, but the pivot away from open weights is drawing fire from the same crowd that championed prior Qwen releases.
