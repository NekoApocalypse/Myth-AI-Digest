## 🧠 Ring-Zero — Scaling Zero RL to 1T Parameters

<https://arxiv.org/abs/2607.12395> (26 points)

Ring-Zero (Ring-2.5-1T-Zero) is a direct test of the "bitter lesson" at scale: take Zero RL — reinforcement learning with verifiable rewards and no human data — and run it at 1 trillion parameters. Three key findings: (1) scaling to 1T significantly improves sample efficiency and performance ceilings; (2) training progresses through a discovery phase followed by a sharpening phase; (3) the model **spontaneously develops** self-verification, parallel reasoning, structured formatting, and even "context anxiety" — without any hand-crafted heuristics.

The paper also proposes a CoT quality evaluation framework across three dimensions (comprehensibility, reproducibility, efficiency), where Ring-Zero shows clear advantages in concise, structured reasoning traces. Competitive on seven math benchmarks.

## 🔍 Detecting LLM Text with Classical ML — 85% Accuracy, Near-Zero False Positives

<https://blog.lyc8503.net/en/post/llm-classifier/> (146 points)

A weekend project that turned out surprisingly effective: the author trained **TF-IDF + LinearSVC** classifiers on ~10,000 human-written texts (pre-2022) vs. LLM-generated counterparts across 7 models (Gemini, Qwen, GLM, Kimi, DeepSeek, Doubao). Seven binary SVM classifiers vote; ≥2 flags = AI-detected sentence.

**Results:** ~85% per-sentence accuracy, ~80%+ F1 across all models. At a 60% detection threshold on full articles, **false positive rate is 0.04%** (tested on 10,000 pre-ChatGPT fanfics). Generalizes to unseen models — Claude Sonnet 4.6 at 71.9%, GPT 5.2 at 73.3%. The entire pipeline runs in-browser as JS (107MB model JSON, 38MB gzipped).

**HN discussion highlight:** Commenters noted the approach independently converges on Pangram's twin-comparison technique, validating that simple statistical features remain surprisingly effective against current-gen LLM outputs.

## 🧪 Libretto — Open-Source Playwright Debugger That Auto-Fixes Tests

<https://libretto.sh/debug-agents> (16 points)

Libretto's PR agent hooks into existing Playwright test suites and triggers automatically on failure: it inspects the live page, diagnoses what changed, and opens a GitHub PR with a proposed fix. Works with any browser provider (local, self-hosted, cloud) — no Libretto runtime lock-in.

MIT-licensed, bring-your-own-model-API-key, no usage fees from Libretto. Currently Playwright-only (Selenium/Puppeteer adapters planned). A clean, focused tool that solves a specific pain point — brittle browser automation tests that break on UI changes.
