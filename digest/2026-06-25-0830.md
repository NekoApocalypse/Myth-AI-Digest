## The Stack

**[Hardware] OpenAI unveils Jalapeño, its first custom inference chip, built with Broadcom** — Not a training chip. Designed specifically for inference workloads with emphasis on perf-per-watt and low operating cost for real-time coding models. OpenAI is now vertically integrating all the way down to silicon. The play: cut Nvidia dependence for serving costs. Early results reportedly beat current alternatives. `<https://techcrunch.com/2026/06/24/openai-unveils-its-first-custom-chip-built-by-broadcom/>`

**[Agents] Computer use is now native in Gemini 3.5 Flash** — No longer a standalone model; screen interaction is a built-in tool inside the main Flash model. Google added targeted adversarial training for prompt injection and enterprise safeguards (user confirmation for destructive actions, automatic injection detection). Ships with a Browserbase demo and open reference implementation. The agent OS race is now a three-way fight. `<https://blog.google/innovation-and-ai/models-and-research/gemini-models/introducing-computer-use-gemini-3-5-flash/>`

**[Agents] GLM-5.2 is the step change for open agents** — 110 points on HN but the most consequential item today. Arena's agent leaderboard has it as the only open model mixing with OpenAI/Anthropic's latest. Matched Opus 4.8 at max thinking effort. The key: it crosses the "Claude Code works without sharp edges" threshold. Fireworks, Together, and every inference provider just got a new flagship. `<https://www.interconnects.ai/p/glm-52-is-the-step-change-for-open>`

**[Frontier Models] Krea 2: SOTA open-weights 12B DiT image model** — Open-weights text-to-image built for aesthetic diversity, not just one polished default. Multi-stage pipeline (pretrain → midtrain → SFT → preference opt → RL). Ships with a prompt expander and style-reference system for creative control. Second place among independent labs on Artificial Analysis. Permissive license, weights on HF. The anti-Midjourney: they actively filter *out* AI-generated images from pretraining to avoid distribution collapse. `<https://www.krea.ai/blog/krea-2-technical-report>`
